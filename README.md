## Instalasi

### Persiapan dependencies
```
- composer install
- copy .env.example, and paste change to .env 
or 
- change name .env.example to .env
```
  
### Generate dan Migration
```
- php artisan key:generate
- php artisan migrate --seed
```

### Persiapan Front End
```
- npm install
- npm run dev
```

### Run Development Server
```
- php artisan serve
```
